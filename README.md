## 📊 Sorting.Visulizer 🕹

---

**_Sorting.Visualizer is a web app for visualizing a bunch of different sorting algorithms Like_**

1. Insertion Sort.
2. Merge Sort.
3. Quick Sort.
4. Heap Sort.

_With the functionality of_ ⏳(**_Processing Speed Control_**) _and_ 📏(**_Array Size Control_**)
